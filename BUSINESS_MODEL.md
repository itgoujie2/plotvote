### Core Philosophy


## üéÅ Free Credit Strategy

### Philosophy
Earned credits are **engagement fuel**, not a replacement for purchases.

**Target Ratio:**
- 70% purchased credits (revenue)
- 30% earned credits (growth/engagement)

**Why This Works:**
1. **Engaged users stay longer** ‚Üí Higher LTV
2. **Creators attract readers** ‚Üí Organic growth
3. **Quality content wins** ‚Üí Self-reinforcing quality
4. **Viral sharing** ‚Üí Lower CAC

**Cost Mitigation:**
- Monthly caps prevent unlimited earning
- Quality thresholds prevent spam
- Delayed redemption (earn now, use later spreads costs)

## üé® Reading Rewards Implementation

### Technical Details

**Tracking System:**
```python
class ChapterView(models.Model):
    chapter = models.ForeignKey(Chapter)
    reader = models.ForeignKey(User)
    created_at = models.DateTimeField(auto_now_add=True)
    read_percentage = models.IntegerField()  # 0-100
    is_unique = models.BooleanField(default=True)

    class Meta:
        unique_together = ('chapter', 'reader')
```

**Credit Award Logic:**
```python
def award_reading_credits(story):
    unique_readers = ChapterView.objects.filter(
        chapter__story=story,
        read_percentage__gte=60,  # Must read 60%+
        is_unique=True
    ).count()

    # Award based on milestones
    if unique_readers == 10:
        award_credits(story.author, 1, 'reading_reward')
    elif unique_readers == 50:
        award_credits(story.author, 5, 'reading_reward')
    elif unique_readers == 100:
        award_credits(story.author, 12, 'reading_reward')
    # ... etc
```

**Monthly Cap Enforcement:**
```python
def can_earn_reading_credits(user):
    current_month_earnings = CreditTransaction.objects.filter(
        user=user,
        transaction_type='earned',
        description__contains='reading reward',
        created_at__month=now().month
    ).aggregate(Sum('amount'))['amount__sum'] or 0

    return current_month_earnings < 50  # Monthly cap
```

## üõ°Ô∏è Anti-Abuse Measures

### Preventing Gaming of Reading Rewards

1. **Unique Readers Only**
   - Track by user account (must be logged in)
   - One credit per unique reader per story
   - IP tracking for suspicious patterns

2. **Quality Thresholds**
   - Must read 60%+ of chapter (time + scroll tracking)
   - Skip rate <30% (users must engage, not bounce)
   - Story must have 3+ chapters (no single-chapter spam)

3. **Monthly Caps**
   - Max 50 credits/month from reading rewards
   - Prevents unlimited earning
   - Ensures purchased credits remain primary revenue

4. **Fraud Detection**
   - Flag bot traffic (rapid sequential reads)
   - Ban users with suspicious read patterns
   - Require email verification to earn credits

5. **Graduated Rewards**
   - First 10 readers = 1 credit (low value for gaming)
   - Exponential rewards for higher milestones
   - Incentivizes quality over quantity
